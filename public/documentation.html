Documentation of DOCumentality

This is a REST API about "documentaries". The application interacts with a database that stores data about different documentaries.

Within "DOCumentality", users will be able to create an account, register and access information about different documentaries, directors and genres. Also, users can update their personal information and create a list of their favorite documentaries within their profile.
